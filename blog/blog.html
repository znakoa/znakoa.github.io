使用 Docusaurus 搭建个人博客网站

Docusaurus 是一个静态网站生成器，它允许你使用 Markdown 和 React 来编写你的文档，然后生成一个网站。

Docusaurus 支持以下功能：

  • 多语言
  • 丰富的主题
  • 丰富的配置
  • 插件系统
  • 丰富的功能

搭建步骤

  1. 安装 Docusaurus CLI

    npx @docusaurus/init@latest init my-website classic

    my-website 是你项目名称，可以随便取

  2. 本地预览

    使用 vscode 打开项目

    cd my-website
    npm run start

  3. 修改配置
